State Information

This topic elaborates on the information to be filled under the State Information section of the calculator.



Field Description
Most Recent Withholding Amount State tax amount withheld in the latest paycheck.
State elected percentage range Each state has its own State tax percentage applied to the total income. This percentage should be selected from the range of available values for the state.
Exempt State Select Yes if the state is exempt from state taxes. There are currently nine states without income tax: Alaska, Florida, Nevada, New Hampshire, South Dakota, Tennessee, Texas, Washington and Wyoming.
Additional State Withholding Enter the amount of additional pending State tax withholdings if any. This amount will get added to the total State tax calculated based on the State taxation rates.
